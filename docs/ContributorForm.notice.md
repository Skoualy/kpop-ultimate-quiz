# 🎵 K-Pop Ultimate Quiz — Guide du formulaire Contributor

Ce guide t'explique comment ajouter ou modifier un artiste dans l'application.

---

## 🚀 Comment accéder au formulaire ?

- Depuis la page **Groupes** → bouton **Ajouter un artiste**
- Depuis la fiche d'un artiste existant → bouton **Modifier**

---

## ① Infos artiste

Commence par renseigner les informations générales de l'artiste.

**Nom de l'artiste** _(obligatoire)_
Saisis le nom officiel de l'artiste. Un identifiant unique est généré automatiquement. Si un artiste avec le même nom existe déjà, un avertissement s'affiche en rouge.

**Statut**
Indique si l\'artiste est actuellement actif ou non.

**Catégorie**
Choisis le type de artiste : Girls group, Boys group, Soliste (F) ou Soliste (M).

> ⚠️ Si tu sélectionnes une sous-unité, la catégorie sera automatiquement celle de l'artiste parent.

**Sub-unit de**
Si cet artiste est une sous-unité d'un autre artiste, sélectionne l\'artiste parent ici. Laisse vide si l\'artiste est indépendant.

> Ce champ est désactivé pour les solistes.

**Année de début**
Saisis l'année de début d'activité. La génération sera suggérée automatiquement.

**Génération**
Pré-remplie automatiquement selon l'année. Tu peux la modifier manuellement.

**Label / Agence** _(obligatoire)_
Saisis le nom du label. Des suggestions apparaissent en tapant. Si le label n'existe pas encore, il sera ajouté automatiquement à la soumission.

**Cover de l'artiste**
Upload une image carrée au format WebP (600×600 px recommandé). Clique sur la zone pour sélectionner un fichier.

Clique sur **Suivant →** quand tous les champs obligatoires sont remplis.

---

## ② Membres

Ajoute les membres de l'artiste.

**Membres actuels** _(au moins un requis)_
Clique sur **+ Ajouter** pour insérer une nouvelle carte membre en haut de la liste.

Pour chaque membre :

- **Nom de scène** _(obligatoire)_ — le nom utilisé dans les jeux
- **Nationalité** — sélectionne avec le drapeau correspondant
- **Rôles** _(au moins un obligatoire)_ — clique sur les badges pour sélectionner un ou plusieurs rôles
- **Portrait** — recommandé en 400×533 px, ratio 3:4

**Anciens membres** _(optionnel)_
Section identique mais pour les membres qui ont quitté l\'artiste.

> Cette section n'apparaît pas pour les solistes et les sous-unités.

Clique sur **Suivant →** une fois au moins un membre actuel avec au moins un rôle renseigné.

---

## ③ Musiques

Ajoute les chansons de l'artiste.

**Title tracks** _(au moins un requis)_
Les titre principaux de l'artiste.

Pour chaque chanson :

- **Titre** _(obligatoire)_ — utilisé comme réponse en blind test, en caractères latins
- **Lien YouTube** _(obligatoire)_ — colle l'URL de la vidéo YouTube. La miniature s'affiche automatiquement.
- **Langue** — Coréen par défaut, Japonais ou Anglais sinon
- **Chanson de début** — active ce toggle si c'est la chanson de début de l'artiste (une seule possible)

**B-sides** _(optionnel)_
Les faces B et bonus tracks.

> Cette section n'apparaît pas pour les solistes et les sous-unités.

Clique sur **Suivant →** une fois au moins un title track avec titre et URL YouTube renseignés.

---

## ④ Export

Dernière étape avant de soumettre !

1. Clique sur **⚡ Générer le JSON** — vérifie que toutes les informations sont correctes
2. Le JSON s'affiche dans la zone de texte — tu peux le relire
3. Clique sur **↓ Télécharger le JSON** pour sauvegarder le fichier
4. Envoie le fichier à l'administrateur pour validation et intégration

Tu peux aussi **📋 Copier JSON** pour le coller directement, ou **✏️ Modifier** pour revenir à l'étape précédente.
