# 🎵 K-Pop Ultimate Quiz — Guide du formulaire Contributor

Ce guide t'explique comment ajouter ou modifier un groupe dans l'application.

---

## 🚀 Comment accéder au formulaire ?

- Depuis la page **Groupes** → bouton **Ajouter un groupe**
- Depuis la fiche d'un groupe existant → bouton **Modifier**

---

## ① Infos groupe

Commence par renseigner les informations générales du groupe.

**Nom du groupe** *(obligatoire)*
Saisis le nom officiel du groupe. Un identifiant unique est généré automatiquement. Si un groupe avec le même nom existe déjà, un avertissement s'affiche en rouge.

**Statut**
Indique si le groupe est actuellement actif ou non.

**Catégorie**
Choisis le type de groupe : Girls group, Boys group, Soliste (F) ou Soliste (M).
> ⚠️ Si tu sélectionnes une sous-unité, la catégorie sera automatiquement celle du groupe parent.

**Sub-unit de**
Si ce groupe est une sous-unité d'un autre groupe, sélectionne le groupe parent ici. Laisse vide si le groupe est indépendant.
> Ce champ est désactivé pour les solistes.

**Année de début**
Saisis l'année de début d'activité. La génération sera suggérée automatiquement.

**Génération**
Pré-remplie automatiquement selon l'année. Tu peux la modifier manuellement.

**Label / Agence** *(obligatoire)*
Saisis le nom du label. Des suggestions apparaissent en tapant. Si le label n'existe pas encore, il sera ajouté automatiquement à la soumission.

**Cover du groupe**
Upload une image carrée au format WebP (600×600 px recommandé). Clique sur la zone pour sélectionner un fichier.

Clique sur **Suivant →** quand tous les champs obligatoires sont remplis.

---

## ② Membres

Ajoute les membres du groupe.

**Membres actuels** *(au moins un requis)*
Clique sur **+ Ajouter** pour insérer une nouvelle carte membre en haut de la liste.

Pour chaque membre :
- **Nom de scène** *(obligatoire)* — le nom utilisé dans les jeux
- **Nationalité** — sélectionne avec le drapeau correspondant
- **Rôles** *(au moins un obligatoire)* — clique sur les badges pour sélectionner un ou plusieurs rôles
- **Portrait** — recommandé en 400×533 px, ratio 3:4

**Anciens membres** *(optionnel)*
Section identique mais pour les membres qui ont quitté le groupe.
> Cette section n'apparaît pas pour les solistes et les sous-unités.

Clique sur **Suivant →** une fois au moins un membre actuel avec au moins un rôle renseigné.

---

## ③ Musiques

Ajoute les chansons du groupe.

**Title tracks** *(au moins un requis)*
Les titre principaux du groupe.

Pour chaque chanson :
- **Titre** *(obligatoire)* — utilisé comme réponse en blind test, en caractères latins
- **Lien YouTube** *(obligatoire)* — colle l'URL de la vidéo YouTube. La miniature s'affiche automatiquement.
- **Langue** — Coréen par défaut, Japonais ou Anglais sinon
- **Chanson de début** — active ce toggle si c'est la chanson de début du groupe (une seule possible)

**B-sides** *(optionnel)*
Les faces B et bonus tracks.
> Cette section n'apparaît pas pour les solistes et les sous-unités.

Clique sur **Suivant →** une fois au moins un title track avec titre et URL YouTube renseignés.

---

## ④ Export

Dernière étape avant de soumettre !

1. Clique sur **⚡ Générer le JSON** — vérifie que toutes les informations sont correctes
2. Le JSON s'affiche dans la zone de texte — tu peux le relire
3. Clique sur **↓ Télécharger le JSON** pour sauvegarder le fichier
4. Envoie le fichier à l'administrateur pour validation et intégration

Tu peux aussi **📋 Copier JSON** pour le coller directement, ou **✏️ Modifier** pour revenir à l'étape précédente.
